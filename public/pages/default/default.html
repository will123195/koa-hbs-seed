  1. All HTML templates are precompiled and can be rendered easily.

    Try pasting this into your browser's javascript console: 
    var html = $.render('pages/default', {title:'Refresh!'});
$('#content').html(html);

  2. All LESS files are compiled into a single css file.

The controller is the context which is rendered:

The filename of this template is: 

{{this.filename}}

The $ indicates a parameter in the controller name. For example, take a look at: 

  1. controllers/pages/@$username.js
    /@me123 
  2. controllers/pages/my-folder/$something.js
    /my-folder/anything
