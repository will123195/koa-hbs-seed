The "build" is automatic when the server starts:

  1. All LESS files are compiled into a single css file.
  2. All HTML templates are precompiled and can be rendered easily.
    Paste this into your browser's javascript console and watch the headline change: 
    var html = $.render('pages/default', {title:'Refresh!'});
$('#content').html(html);

A dollar sign $ in the controller's filename indicates a parameter in the url. For example: 

  1. controllers/pages/@$username.js
    /@me123 
  2. controllers/pages/my-folder/$something.js
    /my-folder/anything

Some convenience methods/objects are available in the controller: 

  1. this.redirect('/')
  2. this.accessDenied()
  3. this.pageNotFound()
  4. this.session
  5. this.ctx
  6. this.params
  7. this.layout
  8. this.view
  9. this.js.push('/my-script.min.js')
  10. this.css.push('/my-styles.css')
  11. this.less.push('/my-styles.less')
